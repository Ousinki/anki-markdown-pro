# Anki Markdown Pro

[English](readme.md) | [中文](readme_zh.md) | [日本語](readme_ja.md)

> Markdownと[Shiki](https://shiki.style)によるシンタックスハイライトをサポートするAnkiアドオン

Markdownでフラッシュカードを作成し、完全な[シンタックスハイライト](docs.md#code-blocks)を利用できます。300以上の言語と60以上のテーマから選択可能で、選択したものだけがダウンロード・同期されます。デスクトップ、モバイル、AnkiWeb全体でライトモードとダークモードをサポートしています。

> [!NOTE]
> [Anki](https://apps.ankiweb.net/) 25.x 以降が必要です。`ツール → アドオン → アドオンを取得` から [`1172202975`](https://ankiweb.net/shared/info/1172202975) を入力してインストールしてください。
> サポートされているすべての機能については、[ドキュメント](docs.md)を参照してください。

- **シンタックスハイライト**: 300以上の言語と60以上のテーマ。選択したものだけが同期されます。
- **高度なコードアノテーション**: 行のハイライト、単語のハイライト、フォーカスモード、エラー/警告マーカーに対応。
- **数式・化学式サポート (MathJax & mhchem)**: 標準的な `$` および `$$` 記法による MathJax 数式レンダリングをネイティブサポート。さらに、追加インストールなしで `\ce{...}` による `mhchem` 化学反応式の描画に対応。
- **フルMarkdown対応**: 太字、斜体、リスト、引用、テーブル、画像の直接貼り付け、アラート、**イコール記号によるハイライト表示 (`==highlight==`)**、**美しい `<kbd>` 3D キーキャップデザイン**など。
- **ルビ・ツールチップ表現 (`[文字]{ポップアップ内容}`)**: 独自のツールチップ構文をサポート。マウスホバーやタップ時に美しい半透明のポップアップを表示し、画面端での自動回り込み配置にも対応。
- **2種類のオーディオ再生モード**: 貼り付けや録音した音声を、自動再生（autoplay）またはクリック再生（スピーカーアイコン表示・iOS等での余白自動非表示）の2つのモードで管理。ツールバーからワンクリックで切り替え可能。
- **Markdown クローズ（穴埋め）カード (Cloze Deletion)**: Markdown クローズカード（`Anki Markdown Cloze` ノートタイプ）をネイティブサポート。CodeMirror エディタ内で公式のクローズショートカット（`Cmd/Ctrl+Shift+C`）や専用のツールバーボタンとシームレスに連動し、スマートなインクリメント穴埋めや同番号穴埋めをサポート。
- **フルキーボードブラウザナビゲーション**: カードブラウザ（Browser）での2軸方向キーナビゲーションに対応。左右キーでカードの切り替え、上下キーでデッキの切り替えが可能。サイドバーの選択移動時に自動的にカードリストを更新します。
- **Browser 一体型カード作成機能 (Inline Card Addition)**: Browser 画面下部に "+ Add Note" ボタンを配置し、右側のエディタパネルで直接新規カードの作成と保存（`Cmd+Enter`対応）をサポート。連続入力が可能で、別ウィンドウを開く必要がありません。選択されているサイドバーのデッキ情報も自動判別されます。
- **洗練されたカードデザイン**: AnkiのネイティブUIに合わせた美しいライト/ダークスタイル。
- **設定パネル**: 言語とテーマを動的に選択可能。
- **クロスプラットフォーム**: デスクトップ、AnkiDroid、AnkiMobile、AnkiWebで動作します。
- **[AIエージェントスキル](#ai-agent-skill)**: [AnkiConnect](https://foosoft.net/projects/anki-connect/)経由でAIエージェントがMarkdownフラッシュカードを作成・管理できる組み込みスキル。

## 使い方

アドオンをインストールした後：

1. **MD** ノートタイプ（追加 → ノートタイプ → MD）を使用して**新しいノートを作成**します。
2. 表面 (Front) にMarkdownで**質問**を書きます。
3. 裏面 (Back) にMarkdownで**答え**を書きます。
4. カードを復習する際、Markdownは自動的にレンダリングされ、シンタックスハイライトが適用されます。

> [!NOTE]
> コードブロック、行のハイライト、アラートなど、サポートされているすべてのMarkdown機能については、[ドキュメント](docs.md)を参照してください。

## ブラウザフルキーボードナビゲーション (Browser Keyboard Navigation)

本アドオンは、Ankiのブラウザウィンドウ向けにマウスを一切使用しない**非常にスムーズなフルキーボードナビゲーション操作フロー**を追加します：

- **マウス不要の初期化**: ブラウザを開くと、フォーカスが自動的にカードリストに移動します。同時に、左側のサイドバーツリーは選択中のカードが属するデッキの階層を自動的に展開してハイライトします。
- **カードリストビュー内**:
  - **左右の矢印キー (`Left` / `Right`)**: リスト内のカードを切り替えます（`Left`で上へ、`Right`で下へ）。
  - **上下の矢印キー (`Up` / `Down`)**: フォーカスが瞬時に左側のサイドバーツリーに移動し、現在のカードが属するデッキが青枠でハイライトされます。
- **左側サイドバーツリー内**:
  - **上下の矢印キー (`Up` / `Down`)**: サイドバーの選択項目（デッキ/タグ/保存された検索）を切り替えます。この際、**カードリストは選択項目の移動に合わせてリアルタイムで自動更新されます**。
  - **左右の矢印キー (`Left` / `Right`)**: フォーカスが瞬時に中央のカードリストに戻り、左右キーでカードのスクロールを再開できます。
- **検索ボックスのクイック脱出**:
  - 検索ボックスに入力中、**`Esc`** キーを押すとフォーカスがカードリストに戻ります。
  - 検索ボックスで **`Enter`** を押すと検索が実行され、実行後 100ms 以内に**フォーカスが自動的にカードリストに戻る**ため、そのままキーボードで検索結果を閲覧できます。

## カードブラウザ一体型カード作成 (Browser-Add Integration)

本アドオンは、カードブラウザ（Browser）内に「カード追加（Add）」パネルをシームレスに統合し、単一ウィンドウでのスムーズな追加・編集体験を提供します：

- **クイック起動**: カードリスト下部にある **`+ Add Note`** ボタンをクリックすると、右側のエディタが新規作成状態になり、`Save Note` と `Cancel` ボタンが表示されます。
- **スマートなデッキ紐付け**: 新規作成されたカードは、左側サイドバーで現在選択されているデッキ（例：`Coding::Figma`）に自動的に紐付けられて保存されます。
- **連続入力とショートカット**:
  - 内容を入力後、**`Ctrl+Enter`** / **`Cmd+Enter`** を押すか、`Save Note` をクリックすることでカードをすばやく保存できます。
  - 保存されると、カードリストが即座に更新され、**エディタは自動的に次の新規カード作成状態になる**ため、キーボード操作のままスムーズな連続入力が可能です。
- **クリーンな下書き消去メカニズム**: 作成中の状態で `Cancel` をクリックするか、**リスト内の他の既存カードをクリック**すると、追加モードが自動的に解除され、データベースから未入力の下書きデータが完全に消去されるため、コレクションが汚れる心配はありません。

## AIエージェントスキル

MarkdownはAIが生成するコンテンツに最適なフォーマットであり、このアドオンはそれを活かしています。AIコーディングエージェント（Claude Code、Codexなど）が[AnkiConnect](https://foosoft.net/projects/anki-connect/)を介してエディタから直接Markdownフラッシュカードを作成・管理できるコンパニオンスキルが同梱されています。AIスキルがコンテンツを生成し、アドオンがそれをレンダリングします。

**前提条件:** [AnkiConnect](https://foosoft.net/projects/anki-connect/)がインストールされ、Ankiデスクトップ版が実行されていること。

インストール：

```bash
npx skills add terkelg/anki-markdown -s anki
```

## 設定

`ツール → アドオン → Anki Markdown Pro → Config`から設定パネルを開きます。

- **言語 (Languages)** — シンタックスハイライトに使用する言語を選択します。新しい言語は保存時にダウンロードされます。フィルターと「選択済みのみ」トグルを使用してリストを管理できます。
- **テーマ (Theme)** — ライトモードとダークモードで別々のShikiテーマを選択できます。
- **UI** — ボーダーレスなデザインにするための「カードレスモード」を切り替えます。
